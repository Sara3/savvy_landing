# Savvy Landing Page

Landing page for **Savvy** — Stop losing money you already earned.

- **Repo**: https://github.com/Sara3/savvy_landing
- **Live**: https://Sara3.github.io/savvy_landing/
- **API**: https://github.com/Sara3/savvy-api | https://savvy-api-1kov.onrender.com

## Auto-deploy

- **GitHub Pages**: Pushes to `main` automatically deploy.  
  Settings → Pages → Source: **main** branch, **/ (root)**.
- **Backend (Render)**: Pushes to `savvy-api` repo auto-deploy on Render.  
  No manual deploy needed.

## Local

- Open `index.html` in a browser, or run a static server.
- Form posts to `https://savvy-api-1kov.onrender.com/api/signup`.

## Structure

- `index.html` — Landing page (deployed to GitHub Pages)
- Backend lives in https://github.com/Sara3/savvy-api (deployed to Render)
